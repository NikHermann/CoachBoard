# CoachBoard
Das Projekt entwickelt für den TSV Ottensheim eine vereinsinterne Plattform zur strukturierten Planung, Dokumentation und gemeinsamen Nutzung von Trainingseinheiten. Ziel ist eine einheitliche Ausbildungs- und Trainingslinie vom Kinder- bis in den Erwachsenenfußball.

(der Projektname kann jederzeit geändert werden)
